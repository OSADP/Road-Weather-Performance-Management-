Projects Included:

RWPM Cloud Azure Hosted Systems:

BSMWorkerRole - Worker role to process incoming connected vehicle basic safety messages
DataProcessorWorkerRole - Worker role that includes the Connected Vehicle processor and the Infrastructure Processor in order to generate INFLO Queue Warnings and Speed Harmonization notifications.
INFLOClassLib - contains support libraries and object types used by the DataProcessorWorkerRole.
InfloWebRole - WebAPI for communicating with the INFLO system
PikalertDataWorkerRole - Worker role for retrieving Pikalert data for use by the RW-PM system
RoadSegmentMappingLib - Mapping library for matching position to mile marker
RWPMPortal - The main web user interface containing the road weather maintenance page, traffic management page, and motorist advisory page
